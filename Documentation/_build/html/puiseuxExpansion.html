puiseuxExpansion module¶

puiseuxExpansion.firstTerms(poly)¶

Using initialTerms(), returns a list of the first terms of the puiseux solotions of poly as puiseuxPoly.puiseux objects.

puiseuxExpansion.initialTerms(poly, positivesOnly=False)¶

Return a list of (gamma,c) pairs giving the first terms of each Puiseux solution of poly.

puiseuxExpansion.recurse(poly, currentMonomial, currentList, bigList, depth)¶

bigList will contain lists of terms for each solution (list of lists). currentList is the one we’re currently computing. Working with lists instead of Puiseux objects for speed and ease of access Assumes currentMonomial is already in the list, and adds the new ones found.

puiseuxExpansion.solutionIterators(poly)¶

* NOT IMPLEMENTED * Returns a list of generators of the Puiseux roots of poly. So solutionIterators(poly)[i].next() can be called repeatedly to show the ith solution of poly.

puiseuxExpansion.solutionList(poly, nterms, asPuiseuxObject=False)¶

Calls recurse() to compute a list of the first nterms puiseux series solutions of poly. If asPuiseuxObject is false, they are returned as lists of (gamma,c) pairs; otherwise they are returned as puiseuxPoly.puiseux objects.
